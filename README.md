# FRONTEND

Frontend  in vue3 and typescript.

## Stack

1. Vue3
2. Typescript

## Requirement

Node 14.15.x

## Running locally in development mode

To get started, just clone the repository and run `npm install && npm run serve`:

    npm install
    npm run serve
